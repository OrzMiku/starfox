## StarFox

A lightweight modpack, with a focus on performance and stability.

This modpack contains a few mods designed to improve the experience, but I've only selected a small number. If you're looking for a more comprehensive modpack, I recommend checking out my other project, LunarFox. You can find more information at the following link: [LunarFox](https://github.com/OrzMiku/lunarfox).
